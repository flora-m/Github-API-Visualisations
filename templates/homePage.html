Weekday Popularity

This is a project for the module CSU33012 Software Engineering which aims to look at the trends in which days are more
popular than others, in terms of number of commits. The authors that contributed on a particular day are also shown.
The focus is especially on larger repositories which are dominated by one language.

Please enter up to 5 addresses of some GitHub repositories into the fields below.
Example of a repository address: nating/cs-exams
